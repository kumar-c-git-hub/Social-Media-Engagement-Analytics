# 📊 Social-Media-Engagement-Analytics Using Python

## 🚀 Project Overview
This project analyzes social media engagement data to uncover patterns that drive user interaction and content performance. Using a dataset of 5,000 social media posts, the project applies data cleaning, exploratory data analysis (EDA), statistical techniques, and visualization to generate actionable insights.

## 🎯 Objectives
- Clean and preprocess raw data  
- Perform exploratory data analysis (EDA)  
- Analyze engagement metrics (likes, comments, shares, watch time)  
- Create a custom engagement score model  
- Visualize trends and patterns  
- Generate business insights  

## 🛠️ Tools & Technologies
Python | Pandas | NumPy | Matplotlib | Seaborn | Plotly  

## 📂 Dataset
The dataset contains 5,000 social media posts with:
- User demographics (age, gender, country)  
- Post details (type, category, hashtags)  
- Engagement metrics (likes, comments, shares, impressions)  
- Sentiment information  

## ⚙️ Project Workflow
- Imported and prepared data using Pandas  
- Cleaned missing values and standardized formats  
- Performed feature engineering  
- Conducted exploratory data analysis  
- Applied statistical analysis (mean, median, variance, percentiles)  
- Built multiple visualizations (scatter, bar, line, heatmap, etc.)  
- Created interactive charts using Plotly  

## 📊 Engagement Score Formula
Engagement Score = (Likes × 4) + (Comments × 3) + (Shares × 5) + (Watch Time × 0.1)

## 📈 Key Insights
- Video content generates the highest engagement  
- Entertainment category performs best overall  
- Positive sentiment posts outperform others  
- Users aged 25–34 are the most active  
- Mobile users show higher interaction and watch time  
- Posting between 12 PM – 3 PM improves visibility  

## 💡 Conclusion
This project demonstrates how data analytics can optimize social media strategies. By focusing on high-performing content types, audience segments, and posting times, engagement can be significantly improved.
